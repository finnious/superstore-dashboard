## Project Overview

This is a learning directory for the **DIVE Cohort 1** — an 8-week course on data, dashboards, and digital work environments (May 12 – June 30, 2026).

## Course Schedule

- **Week 1 (May 12):** Introduction to data, dashboards, and digital work environments
- **Week 2 (May 19):** Understanding KPIs, metrics, and business questions using data
- **Week 3 (May 26):** Basics of data visualization and choosing correct chart types
- **Week 4 (June 2):** Dashboard design principles, layout, and storytelling with data
- **Week 5 (June 9):** Data cleaning, pivot tables, and dashboard creation
- **Week 6 (June 16):** Power BI basics, interactive dashboards, and AI features
- **Week 7 (June 23):** Industry use cases of AI dashboards, using vibe coding for dashboards
- **Week 8 (June 30, tentative):** Graduation
